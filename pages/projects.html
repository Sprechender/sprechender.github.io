preInstall preInstall is a Python script that is designed to run in a command-line environment, offering a user interface made with questionary to automate the installation process of multiple programs.
Similar to winget or apt

stream-tools stream-tools is a command-line tool designed to enhance the Twitch streaming experience.
Developed using the Twitch API, this tool aims to simplify the process of creating clips and managing stream information such as title, category, and more.

py-repo-downloader This python script lets you select using questionary & download specific release assets from a GitHub repo.
